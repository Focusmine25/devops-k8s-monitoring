# DevOps Kubernetes Monitoring Project

## Overview
This project demonstrates a **Kubernetes deployment with monitoring** for a microservice application. It includes a **frontend**, **backend**, **Prometheus**, and **Grafana**, showcasing container orchestration, scaling, and monitoring skills.

---

## Features
- Containerized frontend and backend microservices using **Docker**
- Deployed on **Minikube** (local Kubernetes) using **Helm**
- Monitoring with **Prometheus** and **Grafana**
- Optional **Nginx Ingress Controller** for routing traffic

---

## Technologies Used
- **Docker** for containerization
- **Kubernetes** with Minikube
- **Helm** for managing deployments
- **Prometheus** for metrics monitoring
- **Grafana** for visualization
- **Python/Flask** for backend (example)
- **HTML** for frontend (example)
